Usage - NOT RECOMMENED
  • Step 1: Import the style

Using the CLI:

unistylus add "components/button-all"

Or, include SCSS:

@import '@unistylus/material/components/button-all';

Or, include CSS:

<link rel="stylesheet" href="https://unpkg.com/@unistylus/material-css@0.0.4/components/button-all.min.css">

And, include JS (if available):

<script src="https://unpkg.com/@unistylus/material-js@0.0.4/components/button-all.min.js"></script>
