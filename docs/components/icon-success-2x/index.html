Usage
  • Step 1: Import the style

Using the CLI:

unistylus add "components/icon-success-2x"

Or, include SCSS:

@import '@unistylus/material/components/icon-success-2x';

Or, include CSS:

<link rel="stylesheet" href="https://unpkg.com/@unistylus/material-css@0.0.2/components/icon-success-2x.css">
  • Step 2: Use the class
<i class="icon-success-2x icon-github"></i>
