Usage - NOT RECOMMENED
  • Step 1: Import the style

Using the CLI:

unistylus add "form/form-control-all"

Or, include SCSS:

@import '@unistylus/material/form/form-control-all';

Or, include CSS:

<link rel="stylesheet" href="https://unpkg.com/@unistylus/material-css@0.0.5/form/form-control-all.min.css">

And, include JS (if available):

<script src="https://unpkg.com/@unistylus/material-js@0.0.5/form/form-control-all.min.js"></script>
