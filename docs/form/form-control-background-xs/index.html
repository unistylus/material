Usage
  • Step 1: Import the style

Using the CLI:

unistylus add "form/form-control-background-xs"

Or, include SCSS:

@import '@unistylus/material/form/form-control-background-xs';

Or, include CSS:

<link rel="stylesheet" href="https://unpkg.com/@unistylus/material-css@0.0.2/form/form-control-background-xs.css">
  • Step 2: Use the class
<form>
  <input class="form-control-background-xs" type="text" placeholder="An input ...">
  <input class="form-control-background-xs" type="file">
  <textarea class="form-control-background-xs" rows="3" placeholder="A textarea ..."></textarea>
  <input class="form-control-background-xs" list="datalistOptions" placeholder="A datalist ...">
  <datalist id="datalistOptions">
    <option value="San Francisco">
    <option value="New York">
    <option value="Seattle">
    <option value="Los Angeles">
    <option value="Chicago">
  </datalist>
</form>

<form class="margin-top-2x">

  <input class="form-control-background-xs" type="file" disabled>
